# GUIPilot

GUIPilot 是一个基于视觉理解的 GUI Agent 项目。

它的目标是让 AI 能够观察当前屏幕，并像人一样完成基础电脑操作，比如点击、输入、滚动和按键。

## Project Goal

The goal of this project is to explore a simple visual computer-use agent that can:

- observe the current screen
- decide the next action
- execute basic desktop operations
- complete simple tasks step by step

## Current Focus

The first version is focused on building a minimal loop:

1. capture the current screen
2. analyze the interface
3. decide the next action
4. execute the action locally

## Status

Early stage prototype.
